# Personal portfolio website

## Overview

This project is a portfolio website consisting of:

- Landing (Hero) page with bio and skills section
- Portfolio page with a list of projects
- Contact Page
- Responsive design and layout
- Modern, clean design

The goal of this project is to build a functional website to be used to house my portfolio of work as a junior developer. Ultimately this site will be used to showcase my skills to prospective employers and recruiters. As I progress through this learning journey I may update the site with additional functionality.

## Built With

- HTML5
- Tailwind CSS
